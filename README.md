# Wassup Chat
A real time group chat application is developed where **web socket** is used for client-server connection
- Minimal Front End
- Clean Design

## Technology Used
- NodeJS
- ExpressJS
- HTML CSS & Vanilla Javascript
- Socket.io

## App is deployed on heroku
Access link : https://wassup-chat-x.herokuapp.com/
